# AI/ML Internship – Codomax

This repository contains all the work completed during my 14-day AI/ML internship at Codomax. It covers the full beginner-to-intermediate journey — from setting up a development environment to building, evaluating, and deploying a simple machine learning model.

## 📌 Overview

- **Duration:** 14 Days
- **Focus Areas:** Python, NumPy, Pandas, Data Cleaning, Data Visualization, Machine Learning (Linear Regression)
- **Dataset:** Student Performance dataset (sourced from Kaggle)
- **Final Deliverable:** A working prediction app that estimates a student's score using a trained Linear Regression model

## 🗂️ Repository Structure

```
├── hello.py                       # Day 1  - First Python program
├── Day2_basics.py                 # Day 2  - Python fundamentals
├── Day3_numpy.py                  # Day 3  - NumPy arrays & operations
├── Day4_pandas.py                 # Day 4  - Dataset loading & exploration
├── Student.csv                    # Original dataset (Kaggle)

├── Student_Cleaned.csv            # Cleaned dataset

├── Day7-10_ML_Model.ipynb         # Day 7-10 - Supervised learning, model training,
│                                    predictions, and evaluation (MAE, MSE, R²)
├── Day11-14_Prediction_App.ipynb     # Day 11 - Interactive score prediction app
└── README.md                      # Project documentation
```

## 🚀 How to Run

1. Clone this repository
```
git clone https://github.com/saadshaikhh09/Codomax-aiml-internship.git
cd Codomax-aiml-internship
```

2. Install dependencies
```
pip3 install numpy pandas matplotlib scikit-learn
```

3. Run any script or notebook, for example:
```
python3 hello.py
```
or open `Day11_Prediction_App.ipynb` in Jupyter Notebook / VS Code and run all cells.
